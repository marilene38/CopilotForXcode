# Contributing to GitHub Copilot for Xcode

Thank you for your interest in contributing to GitHub Copilot for Xcode!

## Why Fork Pull Requests Are Closed

At this time, **we are not accepting code contributions via pull requests** from
forks of this repository. Any pull request opened from a fork will be
automatically closed.

This policy exists because the project is maintained by a dedicated team that
manages the development roadmap, security review, and release process
internally.

## Trusted Contributors

In exceptional cases, a project maintainer may add the `review-fork-pr` label
to a fork pull request to allow it to remain open for review. If you believe
your contribution warrants special consideration, please reach out via a
[GitHub Discussion](https://github.com/orgs/community/discussions/categories/copilot)
before opening a pull request.

This project adheres to a [Code of Conduct](./CODE_OF_CONDUCT.md). By
